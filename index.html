Welcome to the XBasic Promotion site.

This site provides a description of the XBasic Programming language and its Program Development Environment (PDE). It also provides a facility ("Issues") for reporting issues and problems to the wider XBasic Community and to receive possible answers and solutions. There are also links to further information and program download sites at the foot of the page.

Introduction to XBasic and PDE

XBasic consists of a Program Development Environment (PDE) and the programming language itself. The language is a flavour of BASIC and is a compiled language that can be compiled and tested within the PDE interactively. Code can also be produced as standalone programs or object libraries.

XBasic is a 32 bit program which will run on 32 and 64 bit PC's operating MS-Windows 10 and earlier and on Linux. XBasic code written for one will operate on the other.

XBasic has extensive file handling capabilities and can incorporate the SQLite C Library for databasing. This makes it particularly useful for utilities development. A library is also available on the Github site for handling json type dynamic objects.

Application developers are provided with a GUI toolkit which enables rapid prototyping of GUI forms and also has a good graphics capability.

To make coding easier, examples of programs are included with the download software.

The XBasic team are also looking for experienced developers who would like to help maintain the software to allow for operating system changes. Please contact the XBUser group on the footer link.
